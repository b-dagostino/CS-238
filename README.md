## Setup

### Pre-requisites

* [Git](https://git-scm.com/)
* [uv](https://docs.astral.sh/uv)

### Installation Steps

1. Clone the repository.

    > `git clone https://github.com/b-dagostino/CS-238.git`

2. Navigate to the project directory:

    > `cd CS-238`

3. Install Python and project dependencies:

    > `uv sync`
